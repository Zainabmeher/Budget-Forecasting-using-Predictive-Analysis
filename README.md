# Budget-Forecasting-using-Predictive-Analysis 
Project Overview

This project analyzes and forecasts departmental expenditure patterns at Olive Sarvodaya Hospital, Hyderabad. The objective is to identify expenditure trends, compare departmental costs, and forecast future spending using historical expenditure data. An interactive Power BI dashboard was developed to provide meaningful insights that support financial planning and resource allocation.

OBJECTIVES 
1.Analyze historical expenditure trends across hospital departments.
2.Compare expenditure patterns among ICU, Radiology, and General Ward departments.
3.Forecast future expenditure using trend-based forecasting techniques.
4.Develop an interactive Power BI dashboard for data visualization and decision-making.

ORGANIZATION
Olive Sarvodaya Hospital, Hyderabad
The study focuses on expenditure data collected from three key departments:

-ICU (Intensive Care Unit)
-Radiology
-General Ward
Data Period: January 2023 – December 2024

TOOLS USED 
Microsoft Excel – Data organization, preliminary analysis, and forecasting calculations.
SQL – Data extraction, querying, filtering, and analysis of hospital expenditure records.
Power BI – Interactive dashboard development and data visualization.
Power Query – Data cleaning, transformation, and preparation.
Linear Trend Forecasting – Used to identify expenditure trends and estimate future spending patterns.
Data Visualization Techniques – Used to present insights through charts, KPIs, slicers, and forecasting visuals. 

FORECAST ANALYSIS
Historical expenditure trends were analyzed using linear trend forecasting techniques. The forecast suggests that departmental expenditure will continue to increase, with ICU expected to remain the highest-cost department. 

CONCLUSION
This project demonstrates the application of data analytics and visualization techniques in the healthcare sector. The dashboard provides a clear understanding of expenditure patterns and supports data-driven financial planning through trend analysis and forecasting.
